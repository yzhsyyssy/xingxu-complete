## 项目说明

本项目将星叙小程序从纯前端项目改造为带有后端API的桌面应用程序。
- 前端：uni-app + Vue 3
- 后端：Vercel无服务器函数
- 部署：GitHub + Vercel
- 桌面应用：Electron
